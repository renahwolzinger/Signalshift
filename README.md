# ShiftSignal

ShiftSignal is a lightweight web app for tracking how AI is affecting jobs, skills, and hiring demand.

## Features

- Searchable AI job impact feed
- Filters for sector and exposure level
- Summary metrics for high-exposure roles, skill shifts, hiring signals, and watchlist matches
- Persistent role watchlist stored in the browser
- Signal mix chart for quick workforce trend scanning

## Run locally

Open `index.html` in a browser. No build step is required.

## Data

The current app uses demo data in `app.js`. A future production version can connect this feed to a news API, labor-market dataset, or curated editorial backend.
